A hands-on 65816 / SNES programming course

Building a Game for the SNES

From an empty folder to a small, playable Super Nintendo game — one concept at a time. Every lesson assembles into a real .sfc ROM you run in an emulator, so you're never just reading. You're building.

ca65 / cc65 Mesen emulator macOS No prior 65816 experience needed

Who this is for

You're comfortable programming and have asar, ca65 (part of the cc65 suite), and the Mesen emulator installed. You don't need prior 6502/65816 or SNES experience — Lesson 2 covers the CPU from scratch.

How each lesson works

A short concept explanation, one fully-commented assembly file, the exact build commands, and what you should see in Mesen. Lessons build on each other — by the end you'll have a running codebase, not disconnected demos.

Course index

This tutorial standardizes on ca65 over asar — mainly because Mesen's debugger has first-class ca65 support, and the reference material this course leans on (nesdoug's series, the Super Famicom Development Wiki) is written for it too.
