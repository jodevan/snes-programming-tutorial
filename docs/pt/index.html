Um curso prático de programação 65816 / SNES

Construindo um Jogo para o SNES

De uma pasta vazia a um pequeno jogo jogável de Super Nintendo — um conceito de cada vez. Cada lição é montada em uma ROM .sfc real que você executa em um emulador — você nunca está apenas lendo. Você está construindo.

ca65 / cc65 Emulador Mesen macOS Sem experiência prévia com 65816

Para quem é este tutorial

Você já programa com desenvoltura e tem o asar, o ca65 (parte do conjunto cc65) e o emulador Mesen instalados. Não é necessário ter experiência prévia com 6502/65816 ou SNES — a Lição 2 cobre a CPU do zero.

Como funciona cada lição

Uma breve explicação do conceito, um arquivo assembly totalmente comentado, os comandos exatos de compilação e o que você deve ver no Mesen. As lições se apoiam umas nas outras — no final, você terá uma base de código funcional, não demos desconectadas.

Índice do curso

Este tutorial padroniza o uso do ca65 em vez do asar — principalmente porque o depurador do Mesen tem suporte de primeira classe ao ca65, e o material de referência em que este curso se apoia (a série do nesdoug, a Super Famicom Development Wiki) também é escrito para ele.
